# VIPPlugin
Rust VIP Plugin that I use on both of my servers. Please do not re-upload or take credit for my work.

Works with ServerRewards Plugin


--Installing--
Drop in plugin folder

DO NOT CHANGE THE NAME!

File will load into Rust server and create a config.

Config can be used for discord webhook!
